# Updating the Dashboard Data

1. Open the Survey123 form and export responses to Excel.
2. Save the Excel file as `SurveyData.xlsx` in the `/code` directory.
3. Open the Power BI Desktop file.
4. Refresh data and confirm relationships on Restaurant Name.
5. Verify pop-up links render correctly after refresh.
6. Re-export the Power BI report to PDF and replace the file in `/powerbi`.
7. Re-publish the report (if using Power BI Publish to Web).
